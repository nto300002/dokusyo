# 📚 読書記録 (Dokusyo)

このリポジトリは、様々な分野の書籍から得た知見をまとめた個人的な読書記録です。各ファイルはブログ形式で整理されており、主要な概念やアイデアを体系的に理解できるように構成されています。

## 📖 カテゴリ別書籍一覧

### 🧠 脳科学

#### [あなたの知らない脳](脳科学/あなたの知らない脳:デヴィットイーグルマン/index.md)
**著者:** デヴィッド・イーグルマン

イーグルマンとホーキンスの知能観の対比、犯罪に対する脳科学的アプローチなど、意識と自由意志についての新しい視点を探求します。

#### [脳は世界をどう見ているのか](脳科学/脳は世界をどう見ているのか:ジェフ・ホーキンス/index.md)
**著者:** ジェフ・ホーキンス

新皮質の予測モデル、1000の知能、格子細胞と場所細胞、知能爆発が起きない理由など、脳の仕組みに基づく知能理論を解説します。
