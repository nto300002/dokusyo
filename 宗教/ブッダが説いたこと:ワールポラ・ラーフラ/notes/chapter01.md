# 四聖諦
## 1.ドゥッカの本質
ドゥッカの例: 恐れ、不安、憎しみ、情欲
無情なものは全てドゥッカであり、一瞬ごとに生起し、消滅する　つまりは流れ続ける
ドゥッカは流れ続けるものであり、その発生と消滅と消滅の過程を観察することが「真理を見る」ことであり、真の幸福を得るために必要なことである

## 2.ドゥッカの生起
### 渇望
- 感覚的喜び、富、権力
- アイデア、考え、意見、理論、概念、信仰
問題は、この利己的な渇望によって生じる

### 生存及び生存の継続
1. 食べもの
2. 感覚器官と外的世界との接触
3. 意識
4. 心的意図あるいは意志

> 心的意図が、存在や継続、増大を行おうとして、それが善悪の行動を起こす
> 意図はカルマである

自らのうちに生まれ、自らのうちに消えていくのがドゥッカの性質

カルマは意図的行為 = 善い/悪い
善い行為が良い結果を生む -> ❌
善い行為はその通りによる結果を生み出す

生を継続しているということを理解したとき、死後もそのエネルギーが持続する
渇望がある限り、継続の輪(輪廻)は継続する

## 3.ドゥッカの消滅
### ニルヴァーナ
ドゥッカが消滅したとき、絶対真理であるニルヴァーナに到達できる
> 言葉を超えた何か

### ニルヴァーナの性質
- 永続的な自己は内にも外にもない = 絶対的真理
- 物事をありのまま見ることができる

> 感覚がないことこそ幸せである

## 4.ドゥッカの消滅への道
- それは「中道」と呼ばれる
### 2つの極端な道を避ける
1. 感覚的な快楽から幸福を得ようとする
> 低俗、通俗的、無益
2. 様々な禁欲的行為で自らを苦しめることによって幸福を得ようとする
> 苦痛を伴い、無益で無価値

### 八正道
1. 正しい理解
2. 正しい思考
3. 正しいことば
4. 正しい行い
5. 正しい生活
6. 正しい努力
7. 正しい気づき
8. 正しい精神統一

> 能力に応じて全てを"同時に"実践する

### これらは3つの基本を増進するための修練
- 倫理的行動
- 心的規律
- 叡智
    - これのみが発達
        - 無情なインテリ
- 慈しみ
    - これのみが発達
        - 心優しい愚か者

#### 慈しみを育む倫理的行動
- 正しい言葉
    1. 嘘をつかない
    2. 輪を乱すような陰口を慎む
    3. 粗暴な言葉、悪意を含む言葉や罵倒を避ける
    4. 無用で役に立たないおしゃべりをやめる
- 正しい行い
道徳的で、安らかな行い <-> 悪事を働く、不誠実な行いをする
- 正しい生活
他人を害することで生計を立てない　<-> 非難されず、尊敬される仕事に就く

#### 心的規律
- 正しい努力
    1. 邪で不健全な気持ちが起こるのを防ぐ
    2. すでに起きた上記のような気持ちを払拭する
    3. 正しく健全な心を起こす
    4. すでに起きた上記のような気持ちを完成させる
- 正しい気づき
    1. 身体の活動
    2. 感覚
    3. 心の動き
    4. 考え、思考、概念
    これらに気を配り、意識する
- 正しい精神統一
マインドフルネス(ジャーナ): 心の動き、その生起と消滅を意識する
    - ジャーナ: 
        第一段階: 欲望、不安、落ち着きのなさが取り除かれ、幸福感に満ちる
        第二段階: 知的活動が抑圧される 平静と心の「１点集中」
        第三段階: アクティブな喜びの感覚がなくなる 残るのは幸せのみ
        第四段階: 全ての感覚がなくなり、純粋な平静と自覚だけが残る

#### 叡智
- 正しい理解
全ての生き物に対し、無私無欲な無執着、愛、非暴力
- 正しい思考
物事をありのままに見る = 四聖諦

### まとめ
四聖諦: ドゥッカ=苦しみ、悲しみ、不完全さ、楽しさ、無常さ
1. それを完全に理解する
2. それを消滅させる
3. それを消滅させたことを体現する(=ニルヴァーナ)
4. ニルヴァーナを体現したまま生き続ける(=死によって消滅)
